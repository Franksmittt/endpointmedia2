PROJECT OVERVIEW
================================================================================
Endpoint Media is a Next.js 14 web application for a Johannesburg-based web
design and digital marketing agency. The site is optimized for local SEO and
designed to generate leads for web design, SEO, and digital marketing services
in the Johannesburg area.

Primary Goal: Dominate Google organic rankings for "web design Johannesburg"
and related local search terms.

Domain: https://www.endpointmedia.co.za


TECHNOLOGY STACK
================================================================================
- Framework: Next.js 14.2.33 (App Router)
- Language: TypeScript
- Styling: Tailwind CSS
- UI Components: Radix UI, Lucide React icons
- Package Manager: pnpm
- Image Optimization: Next.js Image (WebP/AVIF formats)
- Font: Geist (Vercel font family)

TECHNICAL SEO
-------------
✓ Enhanced Metadata
  - Comprehensive title tags and meta descriptions
  - Open Graph tags for social sharing
  - Twitter Card integration
  - Language alternates (en-ZA, en)
  - Canonical URLs on all pages (no trailing slashes)

✓ Schema Markup (JSON-LD)
  - Organization schema
  - LocalBusiness schema (with address, service area)
  - WebSite schema
  - BreadcrumbList schema
  - Article schema (blog posts)
  - Service schema (service pages)
  - FAQPage schema (where applicable)
  - Secure JSON-LD sanitization (XSS protection)

✓ Sitemap & Robots
  - Dynamic XML sitemap (/sitemap.xml)
  - Priority-based URL structure
  - Optimized robots.txt
  - Critical: /_next/ NOT blocked (allows Googlebot to render JS)

✓ URL Structure
  - Clean, SEO-friendly URLs
  - Trailing slash: false (consistent canonicals)
  - WWW redirect: non-www → www (308 redirect)

✓ Core Web Vitals Optimization
  - LCP optimization (priority images for hero sections)
  - Image optimization (WebP/AVIF formats)
  - Proper image sizing and lazy loading
  - Font optimization (system fonts to reduce CLS)
  - Compression enabled
  - Caching headers configured (30-day cache for images)

✓ Security Headers
  - HSTS (Strict-Transport-Security)
  - X-Frame-Options
  - X-Content-Type-Options
  - X-XSS-Protection
  - Referrer-Policy
